# 测试样本数据

本目录包含7条标准测试样本及标注结果，用于校验Skill研判效果。

## 样本目录

- `samples/` - 7条标准测试样本原始数据
- `ground_truth/` - 标准研判结果（人工标注）
- `supplementary/` - 补充分类示例（覆盖7条样本未覆盖的分类）
