# 威胁分类可视化图谱

## 概览

本图谱展示Web威胁分类体系的整体结构，按威胁载体分为三大类：
- 网站数字风险
- 社交媒体数字风险
- APP数字风险

## 分类图谱

```
Web威胁分类体系
├── 网站数字风险
│   ├── 高危
│   │   ├── 钓鱼欺诈（含登录框）
│   │   │   - 定义：冒充机构登录框诱导输入敏感信息
│   │   │   - 核心特征：登录输入框、域名近似、提交至未知服务器
│   │   │   - 辅助依据：浏览器跳转异常
│   │   │   │
│   │   └── 仿冒网站
│   │       - 定义：模仿机构官网或子系统用于欺诈
│   │       - 核心特征：新注册近似域名、含"开户/理财"按钮
│   │       - 辅助依据：域名监控、视觉比对
│   │
│   ├── 中危
│   │   ├── 品牌侵权
│   │   │   - 定义：未经授权使用机构名义从事金融服务
│   │   │   - 核心特征：假投资产品、假保险合同
│   │   │   - 辅助依据：域名监控、卖家渠道比对
│   │   │
│   │   └── 疑似暴露资产
│   │       - 定义：数字资产被意外暴露在公开网络
│   │       - 核心特征：公网IP直接访问、无域名
│   │       - 辅助依据：敏感资产扫描
│   │
│   └── 低危
│       ├── 商标滥用
│       │   - 定义：未授权使用机构商标Logo
│       │   - 核心特征：假"官方合作"诱导
│       │   - 辅助依据：Logo识别、关键词监控
│       │
│       ├── ICP滥用
│       │   - 定义：未备案或滥用金融机构名义备案
│       │   - 核心特征：备案主体与品牌不符
│       │   - 辅助依据：备案号校验

│       ├── 疑似商务合作
│       │   - 定义：合法合作关系引用品牌内容

│       │   - 辅助依据：ICP主体比对
│       │
│       ├── 疑似政务合作
│       │   - 定义：政府机构引用企业内容
│       │   - 核心特征：备案为政府/科研院所
│       │   - 辅助依据：政府域名库比对
│       │
│       ├── 舆论投诉
│       │   - 定义：用户投诉举报负面新闻
│       │   - 核心特征：论坛社区内容
│       │   - 辅助依据：投诉关键词监测
│       │
│       └── 其他
│           - 定义：无法归类、需人工核实
│           - 核心特征：无明显威胁特征
│           - 辅助依据：综合研判
│
├── 社交媒体数字风险
│   ├── 高危
│   │   ├── 疑似仿冒官方账号

│   │   │   - 核心特征：名称头像一致、认证状态
│   │   │   - 辅助依据：官方账号库比对
│   │   │
│   │   ├── 疑似仿冒分支账号
│   │   │   - 定义：假冒地区分行、子公司
│   │   │   - 核心特征：昵称含"分行/营业部"

│   │   │   - 核心特征：博主使用Logo
```

## 风险等级分布

| 风险级别 | 网站数字风险 | 社交媒体数字风险 | APP数字风险 |
|----------|--------------|------------------|-------------|
| 高危 | 钓鱼欺诈、仿冒网站 | 仿冒官方/分支/客服账号 | 仿冒APP、虚假APP |
| 中危 | 品牌侵权、疑似暴露资产 | 品牌滥用账号、仿冒个人账号、推广账号 | （较少） |
| 低危 | 商标滥用、ICP滥用、疑似商务合作、疑似政务合作、舆论投诉、其他 | （较少） | 非官方下载链、品牌滥用、影子资产、多开应用、非官方分发渠道、其他 |
| 未知 | - | - | 其他（APP） |

## 研判要点速查

### 高危分类快速识别

| 分类 | 快速识别特征 |
|------|--------------|
| 钓鱼欺诈 | 有登录框 + 域名近似 + 诱导输入 |
| 仿冒网站 | 新注册域名 + 页面仿官方 + 下载/理财按钮 |
| 仿冒官方账号 | 名称头像一致 + 带"官方"字样 + 认证 |
| 仿冒APP | 图标名称完全相同 + 窃取信息目的 |
| 虚假APP | 公司声明从未开发此APP |

### 易混淆分类快速区分

| 组合 | 区分要点 |
|------|----------|
| 钓鱼欺诈 vs 仿冒网站 | 有无登录框诱导是核心区分点 |
| 商标滥用 vs 品牌侵权 | 前者侧重Logo使用，后者侧重冒充机构服务 |
| 仿冒网站 vs 疑似暴露资产 | 前者有域名仿冒，后者为IP直接访问 |
| 仿冒官方 vs 仿冒分支 | 前者冒充官方主体，后者冒充二级身份 |
| 仿冒APP vs 非官方下载链 | 前者为APP本身伪造，后者为渠道问题 |
