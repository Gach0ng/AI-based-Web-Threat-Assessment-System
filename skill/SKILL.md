## 2. 架构约束

### 2.1 架构边界

```
┌─────────────────────────────────────────────────────────┐
│                     输入数据源                            │
├─────────────────────────────────────────────────────────┤
│ 1. 决策树结果（主判断依据）                              │
│    - 分类结果 + 风险级别 + 决策路径 + 中间节点           │
│                                                         │
│ 2. 云端黑数据（辅助参考）                              │
│    - category_name: 黑类型名称                          │
│    - suggested_level: 黑白灰级别                        │
│    - is_wd_type: 是否网盾维护类型                       │
│                                                         │
│ 3. 威胁情报（辅助参考）                                │
│    - overall_score: 综合评分(0-100)                     │
│    - 各维度细分评分                                      │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    Skill 研判                           │
│  - 决策树结果为主                                        │
│  - 黑数据/情报结果用于增强置信度或发现额外威胁                 │
│  - 输出最终分类和置信度                                  │
└─────────────────────────────────────────────────────────┘
```

### 2.2 权重说明

| 数据源 | 权重 | 说明 |
|--------|------|------|
| 决策树分类结果 | **最高** | 主判断依据，直接影响最终分类 |
| 云端黑数据 | 辅助 | 如检出恶意类型可提升置信度；与决策树矛盾时需研判 |
| 威胁情报 | 辅助 | 综合评分供参考；高评分可作为辅助证据 |

### 2.3 决策树节点说明

决策树中间节点供参考：
- `threat_related`: 威胁情报是否关联
- `icp_empty`: ICP备案是否为空
- `has_brand_keywords`: 是否含品牌关键词
- `has_gray_black_category`: 是否含灰黑产标签
- `has_login_form`: 是否有登录表单
- `js_diff_ratio`: JS开关差异度
- `yolo_ocr_empty`: YOLO/OCR是否为空

## 3. 全分类研判标准

### 3.1 分类定义

#### 仿冒网站（高危）

| 项目 | 内容 |
|------|------|
| 定义 | 模仿机构官网或其子系统（自助登录/贷款申请/理财平台）的网站，用于欺诈、假承诺、数据收集 |
| 判定边界 | 新注册域名与机构官网相似；页面含品牌关键词；高危后缀；境外IP；灰黑产标签 |
| 典型场景 | 假"银行官网"上线，可申请高利率存款或投资产品 |

#### 钓鱼欺诈（含登录框）（高危）

| 项目 | 内容 |
|------|------|
| 定义 | 通过冒充机构或其客服、理财平台登录框，诱导客户输入账号/密码/银行卡信息 |
| 判定边界 | 页面包含登录输入框；域名近似机构官网；检测提交至未知服务器 |
| 典型场景 | 客户收到"银行账户异常"邮件链接至近似域名，输入后被盗 |

#### 商标滥用（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 未授权第三方使用机构商标、Logo、官方名称，以假"官方合作""官方理财产品"诱导客户 |
| 判定边界 | 页面含品牌Logo/名称但主体非机构；含"官方推荐"关键词；ICP备案主体为企业 |
| 典型场景 | 某理财平台使用银行Logo宣称"官方推荐" |

#### 品牌侵权（中危）

| 项目 | 内容 |
|------|------|
| 定义 | 未经授权使用机构名义从事金融服务/产品（如假投资产品、假保险合同） |
| 判定边界 | 平台标题含机构商标但不存在授权；YOLO/OCR非空且含品牌销售信息 |
| 典型场景 | 假"某券商官方理财计划"使用该券商商标 |

#### ICP滥用（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 在中国市场，网站未备案或滥用金融机构名义取得/使用ICP备案号，冒充机构服务 |
| 判定边界 | 网站ICP备案主体非品牌方；域名实际ICP主体与备案信息不符 |

#### 疑似暴露资产（中危）

| 项目 | 内容 |
|------|------|
| 定义 | 品牌或关联组织的数字资产（如IPv4）被意外或恶意暴露在公开/半公开网络 |
| 判定边界 | 直接IP访问的网站；IP归属地为中国但无ICP备案 |

#### 疑似商务合作（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 解析记录含有用户关键词、部署平台关联企业资产，注册域具有企业备案记录 |
| 判定边界 | 域名备案主体与企业有关联；ICP备案主体为政府/事业单位 |

#### 疑似政务合作（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 网站页面含有用户关键字、LOGO等数据关联企业资产，备案属性为政府机构、科研院所等 |
| 判定边界 | ICP备案主体为政府机构/科研院所；页面含品牌关键字 |

#### 舆论投诉（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 公众或者用户通过线上平台多种途径，针对企业提出的质疑、举报、诉求或负面新闻 |
| 判定边界 | 页面内容涉及投诉、举报；平台性质为论坛/社区 |

#### 其他/无害网站（低危）

| 项目 | 内容 |
|------|------|
| 定义 | 无法归类于上述任何分类；无明显威胁特征 |
| 判定边界 | 决策树输出为"其他"；无恶意检出；评分低 |

## 4. 研判逻辑

### 4.1 研判步骤

```
第一步：读取决策树分类结果（主依据）
    ↓
第二步：检查云端黑数据
    ├── 有恶意检出 → 提升置信度或补充证据
    └── 无恶意检出 → 继续研判
    ↓
第三步：检查威胁情报
    ├── 高评分(≥60) → 辅助支撑
    └── 低评分 → 参考
    ↓
第四步：综合判决
    ├── 决策树高危 + 黑数据/情报辅助 → 高置信确认
    ├── 决策树高危 + 黑数据/情报矛盾 → 仔细研判
    ├── 决策树低危 + 无黑数据/情报证据 → 谨慎确认
    └── 决策树低危 + 黑数据高危检出 → 升级研判
    ↓
第五步：输出研判结果
```

### 4.2 研判原则

1. **决策树为主**：决策树分类结果直接决定最终分类，黑数据/情报作为辅助验证
2. **高危优先**：高危分类（仿冒网站、钓鱼欺诈）优先于低危分类
3. **证据链完整**：研判理由需能关联决策树路径 + 黑数据/情报证据
4. **置信度合理**：信息充分时高置信，矛盾或信息不足时降低置信度

## 6. 分类映射

### 6.1 标准分类列表

**高危**：
- 仿冒网站
- 钓鱼欺诈（含登录框）

**中危**：
- 品牌侵权
- 疑似暴露资产

**低危**：
- 商标滥用
- ICP滥用
- 疑似政务合作
- 疑似商务合作
- 舆论投诉
- 其他
- 无害网站

### 6.2 易混淆分类区分

| 分类对 | 区分关键 |
|--------|----------|
| 钓鱼欺诈 vs 仿冒网站 | 钓鱼欺诈特指登录框诱导输入敏感信息；仿冒网站侧重整体模仿官网外观 |
| 仿冒网站 vs 商标滥用 | 仿冒网站整体模仿；商标滥用未授权使用商标Logo |
| 品牌侵权 vs 商标滥用 | 品牌侵权冒充机构从事金融产品服务；商标滥用未授权使用商标 |
| ICP滥用 vs 仿冒网站 | ICP滥用侧重备案信息问题；仿冒网站侧重页面内容模仿 |
| 仿冒网站 vs 疑似暴露资产 | 仿冒网站域名仿冒；暴露资产IP直接访问 |
| 商务合作 vs 仿冒网站 | 商务合作为合法合作关系；仿冒网站为恶意模仿 |
| 政务合作 vs 商标滥用 | 政务合作为政府合作背书；商标滥用为未经授权商业使用 |
